Регистрация / профиль
 
на главную | содержание

Для начала работы с системой необходимо зарегистрироваться, выбрав пункт главного меню "регистрация". При регистрации необходимо правильно заполнить поле "имя команды" (указанть там фамилию и имя для личного участника, название вуза/школы/региона для команды). Поле "логин" должно быть уникальным среди всех зарегистрированных пользователей, поля "пароль" и "пароль (подтверждение)" должны содержать один и тот же пароль.

Остальные поля несут чисто информативную функцию и могут не заполняться.

Для изменения своих регистрационных данных слудует воспользоваться пунктом меню "профиль".
